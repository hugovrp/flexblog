# ✍️ FlexBlog

Projeto criado durante o curso de **CSS Flexbox** da [Origamid](https://www.origamid.com/).  
O site simula um blog com layout fluido e responsivo utilizando exclusivamente Flexbox.

## 💻 Tecnologias Utilizadas

- HTML5
- CSS3
- CSS Grid
- Tipografia Web
